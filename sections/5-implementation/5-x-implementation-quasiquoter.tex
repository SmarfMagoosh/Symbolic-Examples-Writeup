\subsection{QuasiQuotation}
\par As mentioned previously, SymCheck is able to determine if symbolic expressions are malformed at compile time.
It achieves this through the use of a QuasiQuoter. 
A QuasiQuoter is a specialized program in Haskell that overrides the was in which certain expressions are parsed during compilation.
This allows users to work with plaintext representations of symbolic expressions which get parsed into expressions in the core language during compilation.
