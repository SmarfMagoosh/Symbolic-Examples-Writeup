\mypara{QuasiQuotation}
SymCheck accepts symbolic expressions via a \textit{QuasiQuoter}. 
A QuasiQuoter is a Haskell feature that allows adding programmer defined syntax to GHC
(the standard Haskell compiler)~\cite{mainland2007s}.
This allows users to work with textual  symbolic expressions,
which are parsed into the core language during compilation.
SymCheck's quasiquoter has the form \hskin{[ex| ... |]},
where the \hskin{...} are filled in with a symbolic expression.
