\mypara{QuasiQuotation}
As mentioned previously, SymCheck is able to determine if symbolic expressions are malformed at compile time.
It achieves this through the use of a QuasiQuoter. 
A QuasiQuoter is a specialized program in Haskell that allows adding programmer defined syntax to GHC
(the standard Haskell compiler)~\cite{mainland2007s}.
This allows users to work with plaintext representations of symbolic expressions which get parsed into expressions in the core language during compilation.
SymCheck's quasiquoter has the form \hskin{[ex| ... |]},
where the \hskin{...} are filled in with a symbolic expression.
