\begin{enumerate}
    \item $\pathvar$ is our path, or ``context.'' 
    It keeps track of which labels are taking on which values currently.
    
    \item $\env$ is our environment. 
    It functions as a dictionary that maps labels to values. 
    Since a single variable may take on multiple values in different contexts, 
    the environment really label-context pairs to values.
    
    \item 
    
    \item $E[k \rightarrow v]$ inserts the key-value pair $(k, v)$ into dictionary $E$. 
    If $E$ already has an entry for $k$, then this operation updates the entry.
    
    \item $E(\lab)$ looks up the entry for $\lab$ in the dictionary $E$. 
    This operation is only performed if it is known that the dictionary has an entry for $\lab$ 
    
    \item $\lab \in E$ checks if $\lab$ is in the key-set of dictionary $E$
    
    \item $\hastype{x}{T}$ means ``$x$ is of type $T$'' where $T$ is either a base type $\base$, an arbitrary type $\complex$, or. A base type is a type that is not a list, an arbitrary type either a base type or a list of some arbitrary type.     \item $e \gens x$ means ``symbolic expression $e$ generates value $x$''

    \item $x \translate y$ means ``surface expression $x$ translates to  core expression $y$''
    
    \item $xs \append ys$ appends the list $ys$ to the list $xs$''
    
    \item $x \ccons xs$ prepends the element $x$ to the list $xs$''
    
    \item $\fresh{x}$ means that value $x$ is a random value that will never be used again in any other expression
    
    \item $\arbitrary{x}$ means that value $x$ is a random value that may be used again in subsequent expressions.
\end{enumerate}
